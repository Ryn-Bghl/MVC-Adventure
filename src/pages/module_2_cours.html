Chap. Flux d'une requête

Flux d'une requête — l'essentiel

Un utilisateur initie une requête (clic, formulaire, AJAX). Le Controller traite et valide, le Model gère les données, la View présente la réponse (HTML/JSON).

  • Initiation : navigateur → requête HTTP (méthode, URL).
  • Traitement : Controller (validation, authentification).
  • Données : Model (lecture/écriture, logique métier).
  • Rendu : View → réponse HTTP (code + contenu).

Exemple : Blog — Model = base d'articles, Controller = action «Créer», View = page affichant l'article.

Aide‑mémoire : Modèle = cerveau, Vue = écran, Contrôleur = chef d'orchestre.
